Embedding blabs

Blabr — "Scientific computing for the web" — is a tool for interactive computation in the browser (tables, plots, sliders, code, etc.). This page shows how to embed content from Blabr in your own site, and is served (statically) from GitHub.

Presently there are two options: (a) embed a single layout box; or (b), embed a complete blab.


Embed a single layout box


  • Consider the Design a second order PLL loopfilter blab.
  • The content we want to embed is in a single layout box (No. 2) on the right-hand side.
  • The required HTML is:

    <iframe class="noborder" width=100% height=500 src="//blabr.io/?4511714609474df6cfa4&pos=2&logo=0" scrolling="no"></iframe>

  • Note that: blabr.io/?4511714609474df6cfa4 is the blab's URL; 'pos=2' selects the second layout box; 'logo=0' hides the logo.
  • Change the sliders to check the behaviour is the same as the source blab.

Embed a complete blab

  • Consider now the Attractor solutions in scalar-field cosmology blab.
  • Note that 'bare=1' hides the blabs menu items (editing etc.).
  • The required HTML is:

    <iframe width=1070 height=560 src="//blabr.io/?3964da2fb54cf9415555&bare=1"></iframe>
