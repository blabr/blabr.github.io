Embedding scientific computation in your site

Blabr — "Scientific computing for the web" — is a tool for creating interactive tables, plots, sliders, code, etc. in the browser as a blab (web lab). This page shows how to embed a blab in your own site.

Presently there are two options: (a) embed a single layout box; or (b), embed a complete blab.


Embed a single layout box

  • Consider the Design a second order PLL loopfilter blab.
  • The content we want to embed is in a single layout box (No. 2) on the right-hand side of the blab.
  • To embed this content here (as shown on the right), include:

    <iframe width=100% height=500 src="//blabr.io?4511714609474df6cfa4&pos=2" scrolling="no" style="border: none;"></iframe>

  • Note that blabr.io?4511714609474df6cfa4 is the blab's URL; and pos=2 selects the second layout box.
  • Change the sliders to check the behaviour is the same as the source blab.

Embed a complete blab

  • Consider now the Attractor solutions in scalar-field cosmology blab.
  • The required HTML is:

    <iframe width=1070 height=560 src="//blabr.io?3964da2fb54cf9415555" style="border: 2px dotted gray;"></iframe>
